Discord Cleanup Bot

Automated message cleanup for Discord servers. Removes old messages from configured channels on a schedule.

Dashboard

Overview of bot status, last run summary, stats, and run controls. Trigger a full or single-channel cleanup run directly from here.

Config

Edit bot settings — retention period, log level, report frequency, report grouping, and the raw channels.yml file. Preview changes, run a dry run, restore a backup for channels.yml or .env.discord_cleanup, or save immediately without restarting.

Audit

Review the live retention configuration at a glance. See categories, overrides, deep clean channels, exclusions, and report controls in one read-only page.

Schedule

Add or remove scheduled cleanup run times and blackout exceptions. Uses the timezone set via the TZ environment variable, and at least one run time must remain.

Stats

Cleanup statistics in three views — Summary (category totals), Detail (per-channel sorted by count), and Grouped (channels nested under categories). Reset any period from this page after confirming the prompt.

Logs

View the last 200 lines of any log file. Color-coded by level — grey for DEBUG, white for INFO, orange for WARNING, red for ERROR. Auto-scrolls to the bottom on load.

API

All data is also available as JSON at /api/status, /api/stats, /api/channels, /api/schedule, /api/logs/latest, and /api/last_run.
